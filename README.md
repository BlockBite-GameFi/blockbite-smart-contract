## 💎 Project Philosophy

In a landscape filled with speculative tokens and luck-based mechanics, BlockBlast Web3 stands as a beacon of transparency. We operate on a **0-Modal model**:
- **70%** of all ticket revenue goes directly into the weekly Reward Pool.
- **100% On-Chain transparency**: Every prize distribution is verifiable on Solana.
- **Skill-Based Equilibrium**: No house edge. Rewards are distributed from the community to the most skilled players.

## 🎮 Game Mechanics: HDR Ultra Experience

The game features an 8x8 grid logic optimized for high-frequency interaction and stunning visual feedback.

- **9,540 Levels**: A meticulously balanced progression system from "Arcade Casual" to "Hyper-Competitive Cursed Mode".
- **Dynamic Scoring**: Chain multipliers, streak bonuses, and "Perfect Board" alibis (+5000 pts).
- **Pure Code Visuals**: High-fidelity HDR graphics rendered via HTML5 Canvas and Vanilla CSS — zero external assets for sub-second load times.
- **Non-Custodial**: Connect your Phantom, Solflare, or Backpack wallet and play instantly.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- A Solana Wallet (Phantom recommended)
- Some Devnet SOL for transactions

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/nayrbryanGaming/blockblast.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000)
